## 📦 Installation

```bash
curl -fsSL https://github.com/hieudoanm/tg/raw/refs/heads/master/packages/cli/bin/tg -o tg
chmod +x tg
sudo mv tg /usr/local/bin/
```

## 🚀 Usage

```bash
tg
tg message
tg version
tg webhook
```
